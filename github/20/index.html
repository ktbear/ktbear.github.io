如何在github上寻找最适合自己的项目呢？

  1. 首先你可以根据名字搜索你想要的项目
    in:name 项目名

  2. 当你用名字搜索出来时出现几万个项目，这时你不知道如何选择可以再限制星星数量来缩小范围
    stars:>数量

  3. 此时出现的项目还是很多，并且右多种编程语言，这时你可以再次限制语言范围
    language:语言

  4. 你需要找怎样的项目呢，此时可以根据项目描述搜索
    in:description 描述

  5. 你又感觉有些项目太旧了，于是想找一份新的
    pushed:>年份

  6. 还可以根据readme进行搜索
    in:readme 描述
